Magic Ball Program
This is a simple program that simulates a magic 8-ball toy. The program generates a random number between 1 and 9 and uses it to select a response from a list of possible answers. The selected answer is then displayed in a graphical window using the Tkinter library.

Installation
Make sure you have Python 3 installed on your computer.
Download or clone the program files from the repository.
Open a terminal or command prompt and navigate to the directory containing the program files.
Run the program by typing python magic_ball.py or python3 magic_ball.py depending on your Python version.
Usage
Launch the program by running the command python magic_ball.py or python3 magic_ball.py in a terminal or command prompt.
The program will display a window with the text "Magic ball" at the top and a randomly generated answer in the center.
Click the "X" button in the top right corner of the window to close the program.
Customization
You can customize the program by modifying the list of possible answers in the getAnswer function. You can also change the font and color of the text in the graphical window by modifying the arguments of the Label constructor.
